Bird Data Immersive XR System

Unreal-based ecological VR system translating bird data into spatialized audiovisual storytelling.

Project Overview

As a Research Assistant at the University of Auckland's Augmented Human Lab, I developed an immersive virtual reality system in Unreal Engine that transformed real-time bird data into a perceivable ecological experience.

Data Translation

The system converted live 3D bird positions and species information into dynamic visual markers and spatialized birdsong, allowing ecological information to be experienced not just as data, but as an audiovisual narrative unfolding inside virtual space.

Technical System

The project integrated LiDAR point-cloud forest environments, data-driven event triggers, and an Ambisonic spatial audio system. Together, these components created a responsive immersive environment in which environmental data shaped both visual and sonic perception.

Exhibition Value

The resulting work functions as a data-driven ecological storytelling experience and is planned for exhibition at the Len Lye Centre in New Zealand, positioning the project between XR research, digital ecology, and public-facing media installation.
