# 放图片到这里（Images）

把首页/项目页面用到的图片放到这个 `images/` 文件夹里。

## 首页当前用到的文件名

- `emotion-dialogue.jpg`
- `multi-agent-influence.jpg`
- `fireescape-vr.jpg`
- `architecture-portfolio.jpg`
- `3d-modeling.jpg`

如果你想用 `.png`，也可以；我再把页面里的后缀统一改成 `.png` 即可。

## 建议尺寸（更像 Linear 那种“大片”效果）

- 建议比例：16:10 或 16:9
- 建议宽度：1600px 以上（越大越清晰）
